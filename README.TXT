
PROJECT TITLE: Apocalypse Frontier
PURPOSE OF PROJECT: Created 2d scrolling game using java. Added multiple types of actors and levels. Used 3d arrays to store and generate levels and sublevels. Many data structures utilized alongside animation and gravity.
HOW TO PLAY: Use arrow keys to move, avoid the enemies and lava, collect food for points, collect gas canisters and bring to the car to advance to next level. Win the game by getting to the last level and making it to the bunker.
AUTHOR: Armaan Khatwani
